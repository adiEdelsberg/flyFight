# Flying Ship Arena Roadmap

## Phase 1 - Core Flight Prototype

Goal:

A player can control a flying ship.

Tasks:

* Create project structure
* Create FlyingShipService
* Create InputController
* Create ShipConfig
* Create basic camera
* Create mobile controls
* Create controller support
* Create boost mechanic

Success:

Player can fly smoothly on all devices.

---

## Phase 2 - Combat MVP

Goal:

Players can destroy each other.

Tasks:

* WeaponService
* Laser weapon
* Hit detection
* One-hit kills
* Ship destruction animation
* Random respawn

Success:

Players can fight continuously.

---

## Phase 3 - Session Progression

Goal:

Give players immediate feedback.

Tasks:

* Session kills
* Session deaths
* Kill streaks
* Leaderboard
* HUD

Success:

Players care about getting more kills.

---

## Phase 4 - Content Expansion

Goal:

Add gameplay variety.

Tasks:

* New ship classes
* New weapons
* New abilities
* Powerups
* Arena hazards

Success:

Different play styles emerge.

---

## Phase 5 - Retention

Goal:

Increase replayability.

Tasks:

* Daily rewards
* Cosmetics
* Ship skins
* Titles
* Challenges

Success:

Players return regularly.
